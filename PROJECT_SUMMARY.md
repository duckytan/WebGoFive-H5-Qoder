## 🎯 项目概览

### 项目定位
基于 HTML5 Canvas 的单页五子棋应用，支持双人对战、完整禁手规则、存档回放等功能。

### 技术栈
- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **图形**: Canvas API
- **架构**: MVC + 事件驱动
- **开发**: Git, Python HTTP Server

## 🏆 技术亮点

### 架构设计
1. **统一状态管理**
   - `GomokuGame` 核心引擎
   - 单向数据流
   - 模块职责清晰

2. **独立回放引擎**
   - 不影响主游戏状态
   - 支持完整播放控制
   - 可导出回放数据

3. **禁手检测算法**
   - 线性编码统一表示
   - 滑动窗口模式匹配
   - 临时落子无副作用
