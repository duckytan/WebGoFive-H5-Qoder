## 🎯 功能概述

### 核心功能
为H5五子棋新增了**VCF冲四练习模式**，这是一个教学型游戏模式，帮助玩家系统学习掌握VCF（连续冲四）战术。

### 特点
- 📚 类似围棋"死活题"的练习系统
- ✓ 实时验证玩家走法
- 🤖 自动AI防守应对
- 💡 每步都有战术提示
- 📈 进度实时追踪

## 🎮 使用方式

### 进入练习模式
1. 点击"切换模式"按钮，循环切换到"冲四练习（VCF）"
2. 点击"新游戏"按钮
3. 系统随机加载一道练习题

### 练习流程
1. 查看题目标题、描述和提示
2. 在棋盘上点击落子
3. 系统验证是否正确：
   - ✓ 正确 → 自动落下防守子 → 进入下一步
   - ✗ 错误 → 提示正确位置
4. 完成所有步骤后显示庆祝动画

### 操作按钮
- 🔄 **重新练习**：重置当前题目
- ▶️ **下一题**：随机选择新题目

## 💻 技术实现

### 核心类：VCFPracticeManager

**主要方法**
```javascript
getRandomPuzzle()        // 获取随机练习题
validateMove(x, y)       // 验证玩家走法
getNextDefenseMove()     // 获取AI防守位置
advanceStep()            // 前进步骤
isComplete()             // 检查是否完成
getCurrentHint()         // 获取当前提示
getProgress()            // 获取进度信息
```

### 数据结构
```javascript
{
  id: 'puzzle-id',
  title: '题目名',
  difficulty: '入门/中级/高级',
  initialBoard: [...],     // 15×15数组
  solution: [              // 正解序列
    { x, y, player, type: 'attack'/'defense', desc }
  ],
  hints: [...]             // 提示数组
}
```
