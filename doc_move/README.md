## 特别说明

### 关于禁手规则

禁手规则是五子棋的核心复杂度所在，详见 `03_游戏规则与AI算法要点.md` 第2章。包括：
- 三三禁手（一步形成两个活三）
- 四四禁手（一步形成两个四）
- 长连禁手（一步形成六子及以上）

实现细节见 `game-core.js` 中的 `checkForbidden()` 方法。

### 关于AI算法

项目实现了两套AI算法：
1. **经典算法**（game-core.js）：基于评分表 + Alpha-Beta剪枝
2. **深度搜索算法**（ai-advanced.js）：开局库 + 威胁序列 + VCF搜索

详见 `03_游戏规则与AI算法要点.md` 第4章。

### 关于VCF练习模式

VCF（Victory by Continuous Four）是本项目的特色功能：
- 40道经典VCF题库（来自实战定式）
- AI以地狱难度防守
- 自动进度追踪和最佳时间记录
- 支持自由下棋（不限定固定路径）

详见 `01_项目需求规格说明书.md` 第2.5节。
